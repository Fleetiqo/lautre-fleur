## 🔐 Authentication Issue

You got a 403 error because you need to authenticate. Here's how to fix it:

### Option 1: Use GitHub CLI (Recommended)
```bash
# Install GitHub CLI if you haven't
brew install gh

# Login
gh auth login

# Push
git push
```

### Option 2: Use Personal Access Token
```bash
# Push with token
git push https://YOUR_TOKEN@github.com/Fleetiqo/lautre-fleur.git main
```

### Option 3: Use SSH
```bash
# Change remote to SSH
git remote set-url origin git@github.com:Fleetiqo/lautre-fleur.git

# Push
git push
```

### Option 4: Simple Push (if already authenticated)
```bash
git push origin main
```
