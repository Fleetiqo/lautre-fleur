# L'Autre Fleur - Fleuriste à Saint-Hippolyte

Site web moderne pour L'Autre Fleur, fleuriste artisanal situé à Saint-Hippolyte, Québec.

## 🌸 À propos

L'Autre Fleur offre des créations florales uniques et personnalisées pour toutes les occasions. Arrangements sur mesure, livraison locale et cueillette en magasin disponibles.

## 🚀 Déploiement sur GitHub Pages

### Déploiement automatique

1. Créez un repository sur GitHub
2. Poussez le code:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/VOTRE-USERNAME/lautre-fleur.git
git branch -M main
git push -u origin main
```

3. Activez GitHub Pages:
   - Settings > Pages
   - Source: **GitHub Actions**

4. Votre site sera disponible à: `https://VOTRE-USERNAME.github.io/lautre-fleur/`

## 🛠️ Développement local
